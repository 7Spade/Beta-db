# CRM 管理模組 (CRM Management)

此目錄是「客戶關係管理 (Customer Relationship Management)」模組的核心實現，提供完整的合作夥伴、客戶和業務關係管理功能。

## 🎯 功能職責

CRM 管理模組旨在提供一個 360 度的客戶和合作夥伴視圖，集中管理與外部實體（如供應商、客戶、承包商）的所有關係和互動。

## 📁 模組結構

### 👥 核心管理
- **`partners/`**: 合作夥伴管理，包含列表、詳細資料、新增/編輯表單
- **`contacts/`**: 聯絡人管理，管理所有業務聯絡人資訊
- **`overview/`**: 總覽儀表板，提供業務關係的整體視圖

### 📊 業務流程
- **`workflows/`**: 工作流程管理，包含視覺化流程建立器
- **`transactions/`**: 交易記錄管理，追蹤所有業務交易
- **`contracts/`**: 合約管理，管理與合作夥伴的合約關係

### 💰 財務管理
- **`financials/`**: 財務管理，包含收支流程設計和財務分析
- **`performance/`**: 績效評估，追蹤合作夥伴和客戶的表現
- **`compliance/`**: 合規管理，確保所有業務活動符合法規要求

### 📈 分析儀表板
- **`dashboard/`**: 專用儀表板，展示 CRM 相關的統計數據和圖表

## 🏗️ 架構特點

- **模組化設計**: 每個子模組都專注於特定的業務功能
- **統一數據模型**: 共享的客戶和合作夥伴數據結構
- **工作流程整合**: 與其他業務模組的工作流程整合
- **響應式介面**: 支援多種設備的現代化使用者介面

## 🔄 與其他模組的整合

- **合約管理**: 與核心運營模組的合約系統整合
- **專案管理**: 追蹤與客戶相關的專案進度
- **文件管理**: 管理客戶相關的文件和合約
- **財務系統**: 與財務管理模組的數據同步
