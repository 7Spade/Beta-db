# 專案管理模組

## 概述
提供完整的專案生命週期管理，包括規劃、執行、監控和交付，支援多種專案類型。

## 核心功能
- 📋 專案規劃與任務分解
- 🎯 進度追蹤與里程碑管理
- 👥 團隊協作與資源分配
- 📊 專案儀表板與報表
- 🔄 工作流程與審核流程
- 📱 移動端專案操作

## 技術架構
- **前端**: React + TypeScript + Tailwind CSS
- **狀態管理**: Zustand
- **圖表**: Chart.js / Recharts
- **工作流程**: Camunda / Temporal
- **後端**: Node.js + Express
- **資料庫**: PostgreSQL
