# 人力資源管理模組

## 概述

協助企業管理員工資料、薪資福利、績效評估、培訓發展和組織架構，提升人力資源管理效率。

## 核心功能

- 👥 員工資料與組織架構管理
- 💰 薪資結構與福利管理
- 📊 績效評估與KPI追蹤
- 🎓 培訓發展與技能管理
- 📅 考勤與排班管理
- 🏢 職位與職級管理
- 📋 招募流程與人才庫
- 📱 員工自助服務平台

## 技術架構

- **前端**: React + TypeScript + Tailwind CSS
- **狀態管理**: Redux / Zustand
- **圖表**: Chart.js / Recharts
- **工作流程**: Camunda / Temporal
- **後端**: Node.js + Express
- **資料庫**: PostgreSQL
