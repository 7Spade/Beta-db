# 公開頁面路由群組 (Public Route Group)

此目錄 `(public)` 是一個 Next.js 的路由群組，專門用於存放那些不需要使用者登入即可訪問的公開頁面。

## 用途

- **隔離佈局**: 這些頁面通常有自己獨立的、較為簡潔的佈局（例如，只有頁首和頁尾，沒有側邊欄），與需要登入的核心應用程式 `(app)` 佈局分開。
- **組織程式碼**: 將所有公開的、面向訪客的頁面集中管理。

## 檔案結構

- **`layout.tsx`**: 為此群組內的所有頁面提供共享的 UI 佈局，通常包含網站的 Logo、公開導航和頁尾。
- **`about/`**: 關於我們頁面。
- **`blog/`**: 部落格相關頁面。
