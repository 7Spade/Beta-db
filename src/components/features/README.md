# 功能组件 (Features)

包含应用程序的核心业务功能组件。

## 组件列表

- `admin/` - 管理后台组件
- `app/` - 应用核心组件
- `auth/` - 认证相关组件
- `blog/` - 博客功能组件
- `cloud-storage/` - 云存储组件
- `contracts/` - 合同管理组件
- `dashboard/` - 仪表板组件
- `documents/` - 文档管理组件
- `partnerverse/` - 合作伙伴组件
- `quick-actions/` - 快速操作组件
- `settings/` - 设置组件
- `team/` - 团队管理组件

## 开发规范

每个功能模块都包含相关的组件、类型定义和业务逻辑。
