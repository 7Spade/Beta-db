# 智慧文件解析 (DocuParse) 模組

## 概述

此模組是「智慧文件解析」功能的核心，負責接收一個文件路徑，使用 Genkit AI 進行分析，並將提取出的結構化資料（如工料清單）呈現給使用者，最終建立專案和合約。

## 架構設計

本模組遵循清晰的目錄結構，將不同關注點分離：

-   **`actions/`**: 包含與後端邏輯互動的 Server Actions，例如呼叫 AI 流程。
-   **`constants/`**: 定義與文件相關的靜態常數。
-   **`tables/`**: 包含用於顯示和編輯從文件中提取的數據的表格元件。
-   **`types/`**: 存放所有與此模組相關的 TypeScript 類型定義。
-   **`views/`**: 包含此模組的主要頁面視圖元件。

### 核心檔案

-   **`views/docu-parse-view.tsx`**: 功能主頁面，從 URL 讀取文件路徑，處理AI解析，並在成功後呈現結果。
-   **`tables/work-items-table.tsx`**: 一個互動式表格，用於顯示、編輯和驗證從文件中提取出的工料清單。
-   **`actions/docu-parse.actions.ts`**: Server Action，作為前端與後端 AI 流程之間的橋樑。

這種結構確保了程式碼的可維護性、可測試性和可擴展性。
