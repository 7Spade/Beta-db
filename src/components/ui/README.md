# UI组件库 (UI Components)

基于shadcn/ui的基础UI组件集合，提供一致的设计语言和交互体验。

## 组件类型

- **表单组件**: `input`, `form`, `select`, `checkbox`, `radio-group`, `textarea`
- **布局组件**: `card`, `accordion`, `collapsible`, `separator`, `sheet`
- **导航组件**: `breadcrumb`, `tabs`, `sidebar`, `menubar`
- **反馈组件**: `alert`, `toast`, `progress`, `skeleton`
- **数据展示**: `table`, `chart`, `calendar`, `carousel`
- **交互组件**: `button`, `dialog`, `popover`, `tooltip`, `dropdown-menu`

## 使用方式

所有组件都支持TypeScript，遵循无障碍设计原则，可自定义主题和样式。
