# AI功能模块 (AI Features)

集成Google Genkit的AI功能模块，提供智能化的业务处理能力。

## 功能组件

- `flows/` - AI工作流程定义
- `genkit.ts` - Genkit配置和初始化
- `dev.ts` - 开发环境配置

## 工作流程

- **提取工作项流程** - 从文档中智能提取任务信息
- **生成知识条目流程** - 自动创建知识库条目
- **生成技能流程** - 智能分析并生成技能描述
- **生成子任务流程** - 将复杂任务分解为子任务

## 技术栈

- Google Genkit
- TypeScript
- Firebase Functions
