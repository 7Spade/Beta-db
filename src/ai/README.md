# AI 功能模組 (AI Features)

整合 Google Genkit 的 AI 功能模組，提供智慧化的業務處理能力。

## 功能組件

- `flows/` - AI 工作流程定義
- `genkit.ts` - Genkit 配置和初始化
- `dev.ts` - 開發環境配置

## 工作流程

- **提取工作項流程** - 從檔案中智慧提取任務資訊
- **生成知識條目流程** - 自動創建知識庫條目
- **生成技能流程** - 智慧分析並生成技能描述
- **生成子任務流程** - 將複雜任務分解為子任務

## 技術棧

- Google Genkit
- TypeScript
- Firebase Functions
